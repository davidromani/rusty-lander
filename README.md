Rusty Lander v0.1
=================

A Jupiter Lander video game port made with Rust & Bevy game engine.

---

#### Installation requirements

* Git 2.0+
* Rust 1.8+
* Cargo 1.80+

#### Installation instructions

```bash
$ git clone git@github.com:davidromani/rusty-lander.git
$ cd rusty-lander
$ cargo run
```
